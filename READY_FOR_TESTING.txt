                      HOW TO GET & TEST APK
================================================================================

STEP 1: Check GitHub Actions (Takes 10-15 minutes)
   1. Go to: https://github.com/Muhammadkhan2008/DeckKey
   2. Click "Actions" tab
   3. Look for "Build DeckKey APK"
   4. Watch for green checkmark (build complete)

STEP 2: Download APK
   1. Click latest "Build DeckKey APK" run
   2. Scroll to "Artifacts"
   3. Download "DeckKey-Debug"
   4. Extract app-debug.apk

STEP 3: Connect Device & Install
   adb devices
   adb install -r app-debug.apk

STEP 4: Enable Keyboard
   1. Open "DeckKey" app
   2. Tap "Enable in Settings"
   3. Toggle DeckKey ON
   4. Tap "Choose Keyboard"
   5. Select "DeckKey"

STEP 5: Test
   1. Open Termux (or any text app)
   2. Test Ctrl+C (should send CTRL+C)
   3. Test Shift+Arrow (should select)
   4. Test Alt+Tab (should switch)
   5. Try typing in Hindi (हिंदी)
   6. Try typing in Chinese (中文)
   7. Check sound & haptics work
